# 学チャレ合同会社 Landing Page

学チャレ合同会社の公式ランディングページです。
「地域課題を、課題で終わらせない。」をミッションに、地域事業開発室としての役割と提供価値を伝えます。

## 📂 プロジェクト構成

```
.
├── index.html          # メインページ（全セクションを含む）
├── styles.css          # メインスタイルシート（デザインシステム・レスポンシブ対応）
├── script.js           # インタラクション（スクロールアニメーション・ナビゲーション）
└── media/              # 画像アセット（ロゴ、写真、アイコン等）
```

## 🛠️ 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: カスタムプロパティ（CSS Variables）を使用したモダンな設計
  - レスポンシブグリッドレイアウト
  - Apple風のミニマリストデザイン
  - モーショングラフィックスとマイクロインタラクション
- **JavaScript (Vanilla)**: 外部ライブラリ依存なしの軽量実装

## 🚀 ローカルでの実行方法

このフォルダ内で簡易ローカルサーバーを立ち上げることで確認できます。

### Pythonを使用する場合
```bash
python -m http.server 8000
```
ブラウザで [http://localhost:8000](http://localhost:8000) にアクセスしてください。

### VS Codeを使用する場合
拡張機能「Live Server」を使用すると便利です。
